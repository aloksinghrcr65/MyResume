    2. CPU SCHEDULING SIMULATOR

    Description :-

    CPU Scheduling is a process of determing which process will own CPU for execution
    and for what time and in what order while another process is on hold.
    The main task of CPU scheduling is to make sure that whenever the CPU remains
    idle, the OS atleast select one of the process available in the ready queue
    for execution. The selection of process will be carried out by the CPU Scheduler.
    It selects one of the process from memory that are ready for execution.

    In this project, i have use 1 non-preemptive scheduling algorithm i.e.
    First Come, First Serve(FCFS) CPU scheduling Algorithm and 3 preemptive
    scheduling algorithm i.e. One is Shortest Job First(SJF) CPU scheduling,
    second is Shortest Remaining Time First(SRTF) CPU scheduling and third is
    Round Robim(RR) CPU scheduling.

    Language and Technology used :- Java and Swing

    Tools used :- Netbeans
